# Overview - PokeTracker

## Obiettivo

Panoramica generale dell'applicazione PokeTracker. Questo documento contiene informazioni di base sull'app che verranno utilizzate durante lo sviluppo.

## App

PokeTracker è un'applicazione desktop relativa ai Pokemon che funziona come "tracker" per dati sugli allenatori.

L'app sarà molto complessa e sarà un insieme di tante features.
