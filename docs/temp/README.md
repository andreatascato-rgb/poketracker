# File temporanei

## Obiettivo

Questa cartella contiene documenti **temporanei** che non fanno parte della documentazione stabile del progetto: valutazioni, confronti, bozze, analisi ad hoc e simili. Quando non servono più, possono essere eliminati senza impatto sulla documentazione ufficiale.

## Cosa va qui

- Valutazioni di tecnologie, librerie o approcci
- Confronti tra alternative (A vs B)
- Bozze e appunti di lavoro
- Analisi esplorative poi superate
- Qualsiasi file markdown “di passaggio” che non deve entrare negli standard o nel project

## Cosa non va qui

- Documenti che definiscono il progetto (→ `docs/project/`)
- Standard e convenzioni (→ `docs/standards/`)
- Procedure operative (→ `docs/procedures/`)

## Regola

Quando si creano file di valutazione, confronto o analisi temporanea, vanno salvati in `docs/temp/`. Periodicamente si può ripulire la cartella eliminando i file non più utili.
