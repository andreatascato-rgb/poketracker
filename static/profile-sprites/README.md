# Sprite profilo (avatar / personaggio)

Asset pixel per personalizzazione profilo (selettore personaggio). Serviti da `/profile-sprites/` in build.

## Contenuto attuale

### Kenney – Roguelike Characters (CC0)

- **Fonte:** [kenney.nl/assets/roguelike-characters](https://kenney.nl/assets/roguelike-characters)
- **Licenza:** CC0 (uso personale, educativo e commerciale; credito non obbligatorio).
- **Contenuto:** spritesheet 16×16 px con margine 1 px (`Spritesheet/roguelikeChar_*.png`). Più personaggi in un’unica immagine; per l’UI si possono ritagliare singole tile o usare la sheet con coordinate.

## Pack aggiuntivo (download manuale)

**GibbonGL – Gen 2 trainer sprites and portraits** (CC0, stile Gold/Silver/Crystal):

- Pagina: <https://gibbongl.itch.io/gen-2-pokemon-trainer-sprite-and-portraits>
- Download: clic su “Download” (prezzo a 0 €). Itch.io non espone link diretto, va scaricato dal browser.
- Dopo il download: estrarre lo zip in questa cartella (o in una sottocartella, es. `gibbongl-gen2/`) per averlo disponibile in app.

## Uso in app

I file in `static/` sono serviti dalla root: es. `/profile-sprites/Spritesheet/roguelikeChar_transparent.png`.
